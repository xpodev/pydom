Basics
######

This section covers the basics of PyDOM, including the syntax and rendering components.

PyDOM is very minimal but powerful, it only requires understanding 2 concepts to create complex applications:
Creating components and rendering them.

After reading this section, you should be able to create almost any kind of application using PyDOM.
